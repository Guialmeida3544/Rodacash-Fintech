# RodaCash - Projeto Fintech

Tela única desenvolvida para a atividade avaliativa de HTML, CSS e Tailwind CSS.

## Arquivos principais

- `index.html`: página principal do projeto.
- `css/styles.css`: estilos personalizados separados do HTML.
- `docs/LINK_GITHUB_RODACASH.pdf`: PDF com o link público do repositório GitHub.

## Link do repositório

(https://github.com/Guialmeida3544/Rodacash-Fintech)

## Como abrir

Abra o arquivo `index.html` no navegador.

## Observação importante

O projeto usa Tailwind CSS via CDN de CSS e CSS personalizado em arquivo separado. Não há JavaScript próprio no projeto.
